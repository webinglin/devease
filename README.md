# 基础框架(platease)	

> 旨在开发一个基础框架，能够快速的进行其他业务功能，数据分析功能的开发工作

platease框架包含：
1. SpringVMC + Spring + Hibernate
2. 用户，角色，功能菜单，登录认证，日志管理，配置管理  等几个基础功能
3. 采用apache shiro权限认证框架来控制访问
4. 默认的系统桌面，通用的表单验证及转换提交插件，通用的表格插件 等
5. 通用操作日志记录框架-->根据配置将日志信息保存到ES中去 --> 保留可扩展性
